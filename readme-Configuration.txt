Users only need to copy and replace the files of this project to the directory under the Darknet version of YOLOv4, and recompile their own project files for training.

There are two yolov4-custom-orange_1.cfg and yolov4-custom-orange_2.cfg in the CFG directory, the improved YOLOv4 network structure file for this project. 
Among them, the number of anchor boxes in yolov4-custom-orange_1 is 15, and the number of anchor boxes in yolov4-custom-orange_2.cfg is 9, which are the values of anchor boxes calculated based on the citrus data set.
